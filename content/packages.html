OSS Software Packages

GHOST.jl

GHOST.jl, short for GitHub Open-Source Tracker, is a Julia package
for targeted scraping of GitHub user and activity data

tidyorgs

tidyorgs is an R package that detects and standardizes organizations in unstructured text data

diverstidy

diverstidy is an R package designed for detection and standardization of geographic, population,
and diversity-related terminology in unstructured text data
